# Induced Hierarchy Graph

The Induced Hierarchy Graph is a node edge graph that forms hierarchies according to its edge priorities.
## Build

```shell
# build project
yarn
yarn run build

# run tests
yarn run test
```

## Publish

```shell
# Run tests
yarn run test

# Build 
yarn run build

# Tag commit
git tag v0.0.1-rc.3

# Push tags
git push --tags

# NPM dry run
npm publish --dry-run

# Publish
npm publish --access public
```
